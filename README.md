# H14 – Technical Exercise

This repository is a starting point for the H14 technical exercise.

## What you’ll find here
- `PROBLEM.md`: full description of the exercise
- `/backend`: placeholder for the Spring Boot project
- `/frontend`: placeholder for the React project

## How to start
- Fork this repository
- Work on your fork
- Push your commits regularly: is not necessary to keep a perfect history, but we need to see your progress
- When you’re done, share the link with us

Have fun!

## Decisions (optional)
When done you could add a `DECISIONS.md` file to your repository with:

- Assumptions made
- Trade-offs
- Open questions
- What I would change in production
